## 🔍 How to use it well

Use skill as a working reference, not just a reading app.

Try this simple flow:

1. Open the app before you start a UI task.
2. Find the article or note that matches your problem.
3. Read the short guidance.
4. Apply one idea to your screen.
5. Check the result in your design or build.
6. Repeat for the next part of the interface.

This keeps your work focused and helps you make steady progress.

### The app does not open
- Check that the download finished

### Windows asks what to do
- Choose the app file if you see one
